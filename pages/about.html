About Monster Finder (and its creator)

Monster Finder was created by Lily (aka Nova) to provide a service to find what store sells monster for cheapest.
You can request more stores by emailing Lily at nova@novafurry.win, sending a message on telegram @novathefurry or posting an issue on the Github repo novafurry/monsterhunter. If you know how to write JavaScript, and sniff network logs via Inspect Element or otherwise, you can build your own modules by forking and creating a pull request.
Donations are glady accepted and appreciated- however you by no means need to donate.
